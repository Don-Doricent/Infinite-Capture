# InfiniteCapture 🚀

InfiniteCapture is a web project that uses NASA's Astronomy Picture of the Day (APOD) API to fetch and display beautiful space images. Users can endlessly explore the universe by loading new captures.

**🌐 Live Demo:** [infinitecapture.netlify.app](https://infinitecapture.netlify.app/)

## 🌌 Features
- Fetches data dynamically from NASA's APOD API
- Displays daily astronomical images with titles and descriptions
- Responsive and simple user interface
- Infinite scroll to keep exploring more images

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- NASA APOD API
